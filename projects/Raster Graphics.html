What is Raster Graphics?

Raster Graphics is a subset of computer graphics. Raster images comprise of pixels on the screen. If you've taken a digital photograph recently, that photo is a good example of a raster image.

I learnt how to use a Raster Editor such as Photoshop to resize a selfie to 1024px, remove the background, and replace the background. I started with one of my photos:

And edited it by creating a new alpha channel, and using the smart select tool to select the regions without the background.

Then, I inverted the selection so I have the background selected, so I can delete it. I then used the feather tool to blur the edges a bit so the merge will look cleaner. Then, I imported my background image as a layer and aligned it how I wanted. I scaled it to 1024 pixels and then exported it as a png. The end result is below:
