CAD

In this page I will show you what I learned during this module.

What is CAD?

CAD is short for "computer aided design", and is used to create designs digitally, whether it is 2D or 3D. This page will cover the basics of 3D CAD.

Our goal is to create this 3D model of a chess knight, so that we can then 3D print it later on. I will be using Autodesk Fusion 360, although any 3D CAD software can achieve the same result.


To create this model, we're going to use 2 functions, extrusion, and revolution. But lets start with the basics.

Parametric Modelling

In my previous guides, I used the conventional method of dimensioning and modelling. In this section, I'm going to show you an alternative method of modelling called Parametric Modelling.

What is Parametric Modelling?

Paramatric Modelling is a design technique that allows the designer to change the shape and dimensions of their model without editing the dimensions directly. This is very useful when you know the shape of the model you want to create, and makes editing the dimensions fast and easy. Watch the video below to find out to use Paramatric Modelling.

To summarise the video, there are 3 steps to parametric modelling.

  1. Create your model parameters by going to modify > change parameters, and creating them there
  2. Use your parameters to dimension your part
  3. Go back to change parameters to edit the dimensions

Music Box

Now that you know the basics of parametric modelling, let's put it to use by modelling a music box that we can laser cut later on.

This is perfect for parametric modelling because we know the general shape of the model, so we can define certain parameters like length, width and height. We also know the thickness of the plywood we will be cutting it from so we can define that too.
