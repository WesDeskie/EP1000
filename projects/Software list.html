List of softwares

Here is the list of softwares I used throughout the module and why I chose to use them.

Text Editor (Atom)

I chose Atom as my text editor and while I could do with just using notepad, I found that Atom has many useful features that other text editors don't have such as predictive text, GitHub integration, and it's just easy to use in general

Raster Editor (Adobe Photoshop)

I chose Adobe Photoshop because I happen to have some experience with using Photoshop, and more importantly, I also have the student license to use Photoshop.

Vector Editor (InkScape)

I chose InkScape because it is open source, and it is what our lecturer Mr Rodney recommended.

CAD (Fusion 360)

I am very proficient at most CAD software and personally prefer Autodesk Inventor. However, on this occasion, I chose Autodesk Fusion 360 because it is more user friendly which would make my CAD tutorials easier for viewers to understand.

IDE (Arduino)

Throughout this module we will be learning and using Arduino microcontrollers, and therefore it only makes sense to use their IDE.

Browser (Firefox)

I use Firefox all the time because of their privacy and security policies.

Screen capture (Windows snip and sketch)

I use snip and sketch because it is already integrated into Windows 10 and already has a convenient hotkey.

Screen recorder (Xbox Game Bar)

I use Xbox Game Bar for the same reason as snip and sketch. It is convenient and makes recording my tutorials easier.
