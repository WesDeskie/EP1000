Formatting with CSS

After creating your website using HTML, you'll probably want to make it look more aesthetic with some formatting. This is where CSS will come in handy. If you want to learn the basics of HTML, you can click here.

In this webpage, I will be going through the basics of CSS by showing you how I formatted my webpage.

What is CSS?

CSS stands for "Cascading Style Sheet" and is a style sheet language that we will use to format our webpage and get from something like this, to the webpage you see in front of you.

Learn CSS in 12 minutes

In this module, we were encouraged to watch "Learn CSS in 12 Minutes" by Jake Wright. A Youtube video which taught us the basics of CSS in a mere 12 minutes. You can watch his Youtube video here.

I formatted most of my webpage following Jake Wright's video, and added a few extra touches such as the dropdown menu you see at the top of the screen.

So lets look at how I formatted this page. If you want to follow along, you'll need to do a few things.

  • Create a new file called "style.css" in your project folder.
  • Open your text editor, and link your "style.css" to your html file (see below)

Using Divisions

The first thing I did is seperate my content into different divisions using the "<div>" tag and we give them an id, so we can format them later. In my case, I followed Jake Wright's tutorial and created the following divisions.

  • A div called "nav" for my navigation bar
  • A div called "container"
  • A div called "content" within the "container"
  • A div called "main" within "content"
  • A div called "footer"

After that, I placed my content into the "main" division.

Then, I opened up my "style.css" file, and set the width of my container to auto, so it will fill up the width of the browser window. I set the colour to pink, so it will be easier to differentiate the container from the other divs. I repeated the formatting steps for my other divs.

Navigation bar

I formatted my horizontal navigation bar like so. I created an unordered list and then used css to make it horizontal and fixed in place.

Using CSS, I can edit any element I want for example, I formatted the list item by using the id "#nav li" and the links using "#nav a". There's a lot to take in but the main points are that I changed the display of the list items to "inline block" to make them horizontal, and I set changed the position of my "nav" division to "fixed", so it stays at the top of the screen.

Dropdown Menu

By now you should have noticed the dropdown menu appearing when you hover your cursor over the "Learning" tab. I achieved this like so.

  1. Create a div called "dropdown" within "nav"
  2. Use a paragraph tag to write your text
  3. Create div called "dropdown-content" within "dropdown"
  4. Place all your links in the "dropdown-content" div

Conclusion

CSS is a very useful tool in web development as it allows us to come up with cool designs for our websites and is the code responsible for all of the aesthetic webpages you see today. My website only scratched the surface of what CSS is capable of, but I hope my simple tutorial helped you get started on formatting your webpage, or at least led you in the right direction.
